# Ringkasan Undang-Undang RI No. 27 Tahun 2022 tentang Pelindungan Data Pribadi (UU PDP)

## 1. Klasifikasi Data Kesehatan
Data kesehatan dikategorikan sebagai **Data Pribadi Spesifik (Sensitif)**, yang membutuhkan tingkat perlindungan lebih tinggi dibanding data umum.

## 2. Kewajiban Pengendali Data (Fasyankes)
- Menjamin keamanan pemrosesan data pribadi.
- Menunjuk Pejabat Pelindungan Data (DPO) jika melakukan pemrosesan data sensitif dalam skala besar.
- Melakukan penilaian dampak pelindungan data (DPIA).

## 3. Hak Subjek Data (Pasien)
- Hak untuk mengakses data pribadi.
- Hak untuk memperbarui atau memperbaiki kesalahan data.
- Hak untuk mengakhiri pemrosesan, menghapus, atau memusnahkan data pribadi.
- Hak untuk menarik kembali persetujuan (Withdrawal of Consent).

## 4. Pelaporan Kebocoran
Wajib melaporkan kegagalan perlindungan data (kebocoran) kepada subjek data dan otoritas dalam waktu **3 x 24 jam**.

## 5. Sanksi
Pelanggaran terhadap UU PDP dapat dikenakan sanksi administratif (denda hingga 2% dari pendapatan tahunan) maupun sanksi pidana penjara.
